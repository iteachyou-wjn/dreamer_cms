### 模板制作流程

<p>
模板制作需要经过以下几个流程：
</p>

1. 设计页面效果图，一般为PSD或者PNG格式的原图；
2. 将页面效果图输出为HTML格式，后缀名为“.html”；
3. 根据页面内容调用标签生成或者编写标签所需代码；
4. 嵌套标签代码到输出页面对应位置；
5. 测试调试模板文件，保证调用和设计效果一致；
6. 将模板标签、文件上传到模板目录；
7. 在系统设定模板目录，重新生成全站；
