### 模板引擎介绍
<p>
Dreamer CMS从2.0.0版本开始采用了解析式引擎与编译式引擎并存的模式，由于在生成HTML时，解析式引擎拥有巨大的优势，但对于动态浏览的互动性质的页面，编译式引擎更实用高效，Dreamer CMS采用双引擎并存的模式，在保持标签风格一致性的同时，也保证将来开发更多互动模块时有更好的性能。
</p>
